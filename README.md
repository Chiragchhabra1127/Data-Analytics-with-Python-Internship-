# Student Performance Data Analysis

This project is part of my **Data Science/Analysis Internship at Maincrafts Technology**. The goal of Task 1 was to perform Exploratory Data Analysis (EDA) on a student performance dataset.

## 📊 Project Overview
In this task, I analyzed a dataset containing student grades and demographic information. I focused on cleaning the data, calculating key metrics, and visualizing the results.

## 🛠️ Key Features
- **Data Cleaning:** Handled missing values and verified data consistency.
- **Analysis:** - Calculated the average final grade (G3).
  - Identified high-scoring students.
  - Analyzed performance based on gender.
- **Visualizations:** Created histograms to show grade distribution.

## 🚀 Technologies Used
- **Language:** Python
- **Libraries:** Pandas, Matplotlib, Seaborn
- **Tool:** Google Colab

## 📈 Results
- **Average Grade:** 10.42
- **High Scorers (>15):** 40 students
- **Observation:** Male students had a slightly higher average grade in this specific dataset.
